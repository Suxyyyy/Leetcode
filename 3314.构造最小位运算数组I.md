[[3314.构造最小位运算数组I]](https://leetcode.cn/problems/construct-the-minimum-bitwise-array-i/description/?envType=daily-question&envId=2026-01-20)
```cpp
class Solution {
public:
    vector<int> minBitwiseArray(vector<int>& nums) {
        for(int& x : nums) {
            int ans=-1;
            int d=1;
            // 只要x在当前位是1，就继续
            while((x&d)!=0) {
                ans=x-d;
                d<<=1;
            }
            x=ans;
        }
        return nums;
    }
};
```
## 为什么取的是 `x - d`？
因为：
- `d` 是 **最低位连续 1 中的某一位**
- `x - d` 就是把这一位变成 `0`
- 恰好构造出符合 `y | (y+1)` 的最小 `y`

## 为什么有时候结果是 -1？
比如：
`x = 8 = 1000`
第一位就是 `0`，while 根本进不去：
`res 仍然是 -1`
说明：
> **不存在 y 使 y | (y+1) = 8**

检查 `x` 末尾连续的 `1`，  
构造最小的 `y = x - 2^k`，  
使得 `y | (y+1) == x`，  
如果 `x` 末尾不是 `1`，则返回 `-1`。

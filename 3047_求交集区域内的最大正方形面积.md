[[3047.求交集区域内的最大正方形面积]](https://leetcode.cn/problems/find-the-largest-area-of-square-inside-two-rectangles/)
```cpp
class Solution {
public:
    long long largestSquareArea(vector<vector<int>>& bottomLeft, vector<vector<int>>& topRight) {
        int maxEdge=0;
        for(int i=0;i<bottomLeft.size();++i) {
            // 枚举第一个矩形 左下角 右下角
            auto& b1=bottomLeft[i];
            auto& t1=topRight[i];
            // 枚举第二个 j<i 防止后面重复(i,j) (j,i)
            for(int j=0;j<i;++j) {
                auto& b2=bottomLeft[j];
                auto& t2=topRight[j];
                // 宽=右边界-左边界
                int width=min(t1[0],t2[0])-max(b1[0],b2[0]);
                // 高=上-下
                int height=min(t1[1],t2[1])-max(b1[1],b2[1]);
                int edge=min(width,height);
                if(width<=0 || height<=0) continue;
                maxEdge=max(maxEdge,edge);
            }
        }
        return 1LL*maxEdge*maxEdge;
    }
};
```

枚举所有矩形对，计算它们的**交集矩形**，  
最大正方形边长 = `min(交集宽度, 交集高度)`。
**算法流程：**
1. 双重循环枚举矩形对 `(i, j)`，只取 `j < i` 防止重复
2. 计算两个矩形的交集宽度和高度 
	`width = min(x2_1, x2_2) - max(x1_1, x1_2)`
	`height = min(y2_1, y2_2) - max(y1_1, y1_2)`
3. 若无交集，跳过（剪枝）
4. 用 `min(width, height)` 更新最大边长
5. 返回 `maxEdge * maxEdge`

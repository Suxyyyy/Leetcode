[[3507.移除最小数对使数组有序I]](https://leetcode.cn/problems/minimum-pair-removal-to-sort-array-i/description/?envType=daily-question&envId=2026-01-22)
```cpp
class Solution {
public:
    int minimumPairRemoval(vector<int>& nums) {
        int ans=0;
        auto checkSorted=[&](vector<int>& v) {
            for(int i=0;i<v.size()-1;++i) {
                if(v[i]>v[i+1]) return false;
            }
            return true;
        };
        while(1) {
            if(checkSorted(nums)) break;
            int ms=INT_MAX; // 记录最小
            int idx=0; // 最小对应下标
            for(int i=0;i<nums.size()-1;++i) {
                int sum=nums[i]+nums[i+1];
                if(sum<ms) {
                    ms=sum;
                    idx=i;
                }
            }
            nums[idx+1]=ms;
            nums.erase(nums.begin()+idx);
            ++ans;
        }
        return ans;
    }
};
```
**时间复杂度**  $O(N^2)$外层循环最多执行 $N$ 次合并，内层搜索最小和需 $O(N)$ 遍历。
**空间复杂度**  $O(1)$ 。

[[819.最常见的单词]](https://leetcode.cn/problems/most-common-word/description/)

```cpp
class Solution {
public:
    string mostCommonWord(string s, vector<string>& banned) {
        unordered_set<string>ban(banned.begin(), banned.end()); // 禁用词
        string tmp="";
        unordered_map<string,int>m; // 出现频率
        for(char& c : s) {
            if(isalpha(c)) {
                // 忽略大小写
                tmp+=tolower(c);
            }
            else {
                if(!tmp.empty()) {
                    // 过滤禁用词
                    if(!ban.count(tmp)) ++m[tmp];
                    tmp.clear();
                }
            }
        }
        // 若s以字母结尾 tmp是最后一个单词
        if(!tmp.empty() && !ban.count(tmp)) ++m[tmp];
        string ans;
        int maxV=-1;
        for(auto [key,val] : m) {
            if(val>maxV) {
                maxV=val;
                ans=key;
            }
        }
        return ans;
    }
};
```

## 一、题目要点

- 提取 **字母单词**
- **忽略大小写**
- 过滤 `banned`
- 返回 **出现次数最多**的单词

## 二、核心思路

**字符扫描 + 哈希统计**

1. 用 `tmp` 拼接字母
2. 遇到非字母 → 结算一个单词
3. 循环结束后 **补处理最后一个单词**
4. 哈希表中找最大频次

## 五、时间 & 空间复杂度

- **时间复杂度**：`O(n)`
- **空间复杂度**：`O(n)`
